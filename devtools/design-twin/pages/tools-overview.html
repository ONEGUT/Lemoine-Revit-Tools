Setup
Get every linked model current and positioned correctly before anything else in the workflow depends on it: upgrade and link the files, align shared coordinates to a common point, then push that correction back into the link files themselves.

Upgrade & Link Models
Picks Revit files from any folder, lets you choose each file's link placement (Internal Origin, Project Base Point, Center to Center, or Survey Point), upgrades every file to the current Revit version, saves it, and links it into the active model — one file at a time to keep memory flat.
FEEDS Align Coordinates →
12 links across 3 Revit versions all come in current and placed at Survey Point

Align Coordinates
Dummy run ▶
Moves the host Survey Point and/or Project Base Point to a picked grid intersection and level, then rotates and translates each selected link so its same-named grid intersection lands on the host's. Repositions the host's copy of each link only.
FED BY ← Upgrade & Link Models
FEEDS Push Coordinates to Links → Copy from Link → Dimensioning →
Survey Point to A ∩ 1 at Level 1, 3 links aligned

Push Coordinates to Links
Commits an already-aligned link's position into the link file itself: unloads it, opens it standalone, moves its own Project Base Point/Survey Point to match, saves in place, then publishes coordinates and re-places the link via Shared Coordinates.
